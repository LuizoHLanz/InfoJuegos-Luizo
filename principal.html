REQUISITOS PARA JUGAR

Requisitos para jugar Minecraft

Se necesita tener un PC con un SO Windows 7 o superior potente y bueno para poder jugar.

Se requiere tener este tipo de componentes en tu PC :

  • CPU: Intel Pentium D | AMD Ahtlon 64 2,6 GHz.
  • GPU: Intel HD Graphics | NVIDIA GeForce 9600 GT | AMD Radeon HD 2400.
  • Memoria RAM: 2 GB.
  • Sistema Operativo: Windows 7 en adelante.
  • Espacio en disco: 200 MB.
  • Versión de Java: Java 6 Release 45.

Requisitos para jugar Fortnite

Fortnite está diseñado para funcionar bien en una gran variedad de hardware para PC , de modo que la mayor cantidad posible de jugadores pueda subir al autobús de combate y jugar.
Para disfrutar de Fortnite de la mejor manera en PC , recomendamos las siguientes especificaciones del sistema.

Especificaciones del Sistema

  1. Tarjeta gráfica: Nvidia GTX 960, AMD R9 280 o DX11 GPU equivalente
  2. Memoria gráfica: 2 GB de VRAM
  3. Procesador: Core i5-7300U de 3.5 GHz, AMD Ryzen 3 3300U o equivalente
  4. Memoria: 8 GB de RAM o superior
  5. SO: Windows 10 de 64 bits
